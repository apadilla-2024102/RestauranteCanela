# RestauranteCanela

Sistema backend para la gestión de un restaurante desarrollado con Node.js y Express.

---

## 📌 Descripción

**Restaurante Canela** es un proyecto que permite administrar diferentes funciones de un restaurante como pedidos, menú, pagos y reservas.

Está organizado en forma de **microservicios**, lo que facilita su mantenimiento y escalabilidad.

---

## ⚙️ Tecnologías

- Node.js
- Express.js
- JavaScript
- Nodemailer
- CORS

---

## 🧩 Funcionalidades

- 🛒 Gestión de carrito
- 📋 Administración del menú
- 🧾 Creación de pedidos
- 💳 Procesamiento de pagos
- 📅 Reservas
- ⭐ Reseñas
- 🎟️ Promociones
- 📊 Reportes
